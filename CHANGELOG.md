# Changelog

## v2.5.1 — 2026-07-14

### UI修正

- ライブラリ内タブが保存キューに押し潰され、上半分が切れる問題を修正
- 保存キューとオフライン容量サマリーを作品一覧と同じスクロール領域へ統合
- ライブラリ内タブを縮まない固定高さへ変更

## v2.5.0 — 2026-07-14

### 探す・フォロー

- フォロー新着フィード、作者フォロー・解除、前回確認後のNEW表示を追加
- 作者プロフィール内検索、タグ・シリーズ絞り込み、作者全作品の一括保存を追加
- 作者／タグのミュートと解除管理を追加
- 文字数、最低ブックマーク数、投稿期間、R-18、AI生成、シリーズ種別、読了除外の高度検索を追加
- フォロー新着と購読シリーズ新話の任意ローカル通知を追加

### 読書

- 文字サイズ、行間、文字の太さ、段落間隔、余白、縦書き列間を連続調整可能に変更
- 明朝体／ゴシック体、明るさ固定、ステータスバー非表示、本文タップによるツールバー開閉を追加
- 本文長押しハイライト、任意範囲の引用取り込み、色・メモ・コピー・共有を追加
- 引用一覧から保存位置へ直接戻れるように改善
- 読了時にシリーズ次話を自動で開く設定を追加
- 作品URLの直接コピーと共有を分離
- コメント・リアクション用のPixiv作品ページをアプリ内WebViewで表示

### シリーズ・オフライン

- シリーズ各話の未読・進捗・読了・保存済み表示と次の未読話導線を追加
- シリーズ全話・作者全作品・複数選択作品の永続ダウンロードキューを追加
- Wi-Fi限定、本文のみ、挿絵込み、読了後自動削除を追加
- 強制終了後の再開、失敗作品だけの再試行、保存中の状態表示を追加
- 購読シリーズの新話を起動時・アプリ復帰時に自動保存する機能を追加
- オフライン総容量と作品ごとの概算容量を表示

### ライブラリ・統計

- 引用一覧、ミュート一覧、保存キュー管理をライブラリへ追加
- 複数選択で削除、読了／未読変更、本棚追加・移動、オフライン保存に対応
- 本棚内の手動順序変更を追加
- 読了日時を記録・表示
- 1日／週間読書目標、現在・最長の連続読書日数を追加
- よく読む作者・タグの集計を追加
- 1日1回・最大7世代の自動バックアップと復元前プレビューを追加

### Android・配布

- Pixiv作品URLのApp Link、カスタムスキーム、Android共有Intent受信を追加
- GitHub Releaseの更新確認、現在バージョン、変更履歴表示を追加
- 新しいDBテーブルと設定をバックアップ／復元対象へ追加

### 補足

Pixiv APIクライアントには小説コメントの取得・投稿・リアクション操作APIがないため、これらはログインCookieを共有したPixiv公式作品ページをアプリ内WebViewで表示して利用します。
